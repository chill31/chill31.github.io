Basic Tags

Hi, welcome!
In this page, you will get started in html by learning a few tags used in websites.






"Title" Tag

This tag is used to give a title to the webpage, like, if you look up in your browser, you see a heading "Basic HTML Tags", this is done with the help of this tag.





"p" Tag

This tag is used to define a paragraph in the webpage, see the example below





"img" Tag

This tag is used to show an image in the webpage.





"input" Tag

This tag is used to take input from a user, it has many types, like date, color, button and more.





"div" Tag

These tags are used for containing things, like tables, and inputs, etc.





"ul" Tag

These tags are used for listing, it has another tag, "ol" ( explained below ), "ul" means un-ordered list ( bulleted ), and "ol" means ordered list ( numbered )





"footer" Tag

These tags are used to give a footer ( positioned in the bottom ), mostly used for credits and rights.
