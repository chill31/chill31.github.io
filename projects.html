Chill31 Blog

Next.js dynamic functions advanced

Built with Next.js, this blog app provides advanced functionality with error checking. There is user authentication but the website works without logging in. There are administrators who can manage (create, edit and delete) blogs. A blog can either be public or private, setting whether all people can see the blog or only logged-in users can see the blog. Markdown is supported in the blog content.

HTML5 number input utility

A single javascript file to easily implement strict behaviour on all number inputs. Many browsers do not have strict number and even allow alphabets and symbols. This file disables everything except decimals and integers.
